What is CSS Specificity?

It's about winning. In other words which CSS rule has higher specificity (wins) and is applied to an element.

Let's say you have the following html

<ul>
  <li>TDD</li>
  <li>BDD</li>
  <li class="selected">PDD</li>
</ul>

What is the color of PDD based for the following CSS?

.selected { color: red; }
ul li { color: black; }

How is it calculated?

It's a point system. Each selector is worth one of the following number of points:

  • 1 point for element selectors (h1) and pseudo-elements (::before)
  • 10 points for class selectors (.selected), attribute selectors ([type="radio"]) and pseudo-classes (:hover)
  • 100 points for id selectors (#selected)

Special Cases!

  • What if it's a tie or the same selector? Last one wins.
  • What about inline styles? They always win (1000 points).
  • What about !important? Actually !important always win.
  • What about the universal selector *? It never wins (0 points).
